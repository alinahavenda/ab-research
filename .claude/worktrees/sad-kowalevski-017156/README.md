# A/B UX Research — Bus Ticket Booking

Дипломний проект: **"Вплив UI на поведінку користувачів при купівлі автобусних квитків"**

## Про проект

A/B тестування двох версій інтерфейсу сервісу купівлі автобусних квитків.
Мета — дослідити як UI рішення впливають на поведінку користувачів і успішність покупки.

## Версії інтерфейсу

| Версія | Кроків | Ключові відмінності |
|--------|--------|---------------------|
| A | 5 | Стандартний інтерфейс — перевантажені картки, окремий екран вибору місця, 7 полів форми, без прогрес-бару |
| B | 4 | Оптимізований інтерфейс — спрощені картки, вибір місця вбудований у картку, 4 поля, прогрес-бар |

## Гіпотези

| ID | Гіпотеза |
|----|----------|
| H1 | Менше кроків → більше завершених покупок |
| H2 | Зрозумілі кнопки → менше помилок |
| H3 | Менше інформації → швидше рішення |
| H4 | Прогрес-бар → більше довіри |
| H5 | Сукупний ефект H1–H4 |

## Технічний стек

- **Frontend:** React 18 + Vite + CSS Modules
- **Backend:** Python 3.11 + FastAPI
- **Database:** Supabase (PostgreSQL)
- **Hosting:** Vercel + Render
